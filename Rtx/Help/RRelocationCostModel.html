Relocation Cost Model

Class Name

  • RRelocationCostModel

Location in Objects Pane

  • Models > Model > Consequence > Economic > Relocation Cost

Model Description

Model Form

    $$D=\sum\limits_{i}{{{\rho }_{i}}\cdot }\left( 1-{{\varsigma }_{i}} \right)$$ $$C=c \cdot D \cdot t $$ Note: Since this model is based on time, the "Cost" response, i.e. \({C}\), cannot be used for other Sampling Analyzers, except for the Discrete Event Sampling

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Time

  • \({t}\) = Time object

Population List

  • \({\rho_i}\) = Number of occupants of building \({i}\)

Repair Or Restoration Status List

  • \({\varsigma_{i}}\) = Status of Repair or Restoration of building \({i}\) (1 if completed, 0 otherwise)

Relocation Cost Per Day Per Person

  • \({c}\) = Cost of relocating and providing temporary shelter for one person for one day

Output

  • \({D}\) = Total number of people who need to be relocated and provided temporary shelter
    • The output is an automatically generated generic response object, which takes the object name of the model plus “DemandResponse”.
  • \({C}\) = Total cost of relocating and providing temporary shelter
    • The output is an automatically generated generic response object, which takes the object name of the model plus “CostResponse”.

Right-click Menu

Remove

  • Removes the object.
