Time Event Type 1

Class Name

  • RTimeEventType1Model

Location in Objects Pane

  • Models > Model > Event > Time Type 1

Model Description

Model Form

  • This model schedules the events in some specified points in time, input by the user.
    • Note: At those specified points in time, the current value of the input parameters are modified, based on the modification rule defined by the user.

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Time Parameter

  • Time object

Event Time List

  • List of the time of the events which are supposed to be scheduled.

Parameter List

  • List of parameters which are supposed to be modifed at the time of the scheduled events.

Modification Rule

  • Method of modifying the current value of the parameters:
  • The “Add” option, adds the current value of corresponding modification factor to the current value of the parameter.
  • The “Substract” option, substracts the current value of corresponding modification factor from the current value of the parameter.
  • The “Multiply” option, multiplies the current value of corresponding modification factor to the current value of the parameter.
  • The “Divide” option, divides the current value of the parameter by the current value of corresponding modification factor.
  • The “Set Equal To” option, sets the current value of the parameter to the current value of corresponding modification factor.

Modification Factor List

  • List of the modification factors. This size of this list must be either one or equal to the size of the list of parameters.

Output

  • This model does not generate response objects.
  • It provides the scheduled events to the orchestrating algorithm.

Right-click Menu

Remove

  • Removes the object.
