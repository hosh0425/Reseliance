FEMA-NIBS Bridge Damage Model

Class Name

  • RFemaNibsBridgeDamageModel

Location in Objects Pane

  • Models > Model > Consequence > Economic > FEMA-NIBS Bridge Damage

Model Description

Model Form

  • This model produces the damage ratio and the percent functional for a bridge subjected to the ground shaking and the permanent ground deformation based on the HAZUS (FEMA-NIBS 2003) approach.
  • For further information on this model, refer to FEMA-NIBS (2003).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Analysis Type

  • List of the types of analysis including ground shaking, ground failure, or both

Bridge Type

  • Type of the bridge according to FEMA-NIBS

Sa 1

  • \({S_{a_1}}\) = 5%-damped elastic spectral acceleration at the period of 1.0s at the bridge site

Sa 2

  • \({S_{a_2}}\) = 5%-damped elastic spectral acceleration at the period of 0.3s at the bridge site

Time

  • Number of days elapsed since the earthquake event

Number of Spans

  • Number of Spans (given as an integer)

Angle of Skew

  • Angle between the centerline of a pier and a line normal to the roadway centerline (given as a real number)

PGD

  • \({PGD}\) = Permanent ground deformation

Bridge Length

  • Bridge Length (given as a real number)

Span Width

  • Span Width (given as a real number)

Model Uncertainty

  • A standard-normal random variable

Output

  • Damage Ratio
  • Percentage of Functionality
  • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.

References

  • FEMA-NIBS (2003) Earthquake loss estimation methodology—HAZUS technical manual. Federal Emergency Management Agency and National Institute of Building Sciences, Washington, DC, USA.
