Water Station Functionality

Class Name

  • RWaterStationFunctionalityModel

Location in Objects Pane

  • Models > Model > Infrastructure > Water > Water Station Functionality

Model Description

Model Form

    $$F=\left\{ \begin{matrix} 0 & ; & \left( \eta >0.1 \right)\text{OR}\left( d>1 \right) \\ 1 & ; & \text{Otherwise} \\ \end{matrix} \right.$$

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Water Station Information Model

  • The associate Water Station information model

Damage Ratio

  • \({\eta}\) = damage ratio of the Water station

Damage State

  • \({d}\) = damage state of the Water station. Values of 0 to 4 are corresponded to damage states none to complete.

Output

  • \({F}\) = Functionality status of the Water station (1 if functional, 0 otherwise). Once the scheduled repair operation is completed, the Water station becomes functional.
    • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.
    Note: Functionality features the performance of internal components of the water station.

Right-click Menu

Remove

  • Removes the object.
