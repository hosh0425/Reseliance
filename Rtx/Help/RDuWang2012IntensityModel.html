Du Wang 2012 Intensity Model

Class Name

  • RDuWang2012IntensityModel

Location in Objects Pane

  • Models > Model > Hazard > Earthquake > Intensity > Du Wang 2012 Intensity

Model Description

Model Form

  • This model produces the cumulative absolute velocity at specified locations for given magnitude and hypocenter location of several earthquake sources as input , based on the Du-Wang (2012) attenuation relation.
  • Depth of each earthquake is predicted based on the magnitude.
  • For more information, see Du-Wang (2012).

DDM Sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Magnitude List

  • Magnitudes of various earthquake sources

Hypocentre Location List

  • Hypocenter locations of earthquake sources, which automatically will yield the radius \({R}\) to the various output locations.

Interevent Sigma

  • Inter-event model error, typically a standard normal random variable

Intraevent Sigma

  • Intra-event model error, typically a standard normal random variable

Fault Type

  • Fault mechanism that can either be Unspecified, Normal, ReverseOblique, or Reverse.

Response Type

  • Type of the response is \({CAV}\).

Structure Location List

  • List of the locations where the intensity will be computed at them (the output will give as many intensity values as the locations provided here)

Site Class

  • Site class that can either be UnspecifiedClass, SGSSiteClassC, SGSSiteClassD.

H and C1 to C8 Uncertainty

  • The uncertainty in attenuation relation coefficients represented by Standard Normal random variables.

Output

  • Earthquake intensities (as many as the locations provided in the input)
  • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.
