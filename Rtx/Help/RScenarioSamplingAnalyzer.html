Scenario Sampling Analyzer

Class Name

  • RScenarioSamplingAnalyzer

Location in Objects Pane

  • Methods > Analyzer > Scenario Sampling

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Random Number Generator

  • List of defined random number generators

Transformer

  • Indicates the transformer object which transforms random variables in original space to the standard normal space or vice versa

Accumulator

  • List of defined accumulators

Function List

  • List of defined functions to analyze

Function Accumulation Type List

  • 0 to aggregate data, 1 to calculate the maximum of data, 2 to calculate the minimum of data and 3 to consider the last data during scenario sampling.

Threshold List

  • List of thresholds

Occurrence Model List

  • List of defined occurrence models
  • The standard unit of lifetime is defined in terms of days.

Timespan

  • Life time
  • The standard unit of lifetime is defined in terms of days

Time Parameter

  • List of defined time parameters

Output

  • The lifetime probability of failure

Right-click Menu

Remove

  • Removes the object.

Run Analysis

  • Runs the scenario sampling analyzer.
