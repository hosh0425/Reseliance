Function Evaluation Analyzer

Class Name

  • RFunctionEvaluationAnalyzer

Location in Objects Pane

  • Methods > Analyzer > Function Evaluation

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Function

  • List of the limit-state functions to evaluate

Evaluate Gradient

  • Determines whether to evaluate gradient.

Set Random Variables To Mean

  • Determines whether to set random variables to mean.

Print Random Variable List

  • Determines whether to print ranodm variables' list.

Plot Model Flowchart

  • Determines whether to plot model flowchart.

Hint:

  • In case of using Rt's interface for external software, using this object comes useful when trying to check whether the two software are well-connected.

    • Output

    • The value of the function
    • Model flowchart

    Right-click Menu

    Remove

    • Removes the object.

    Run Analysis

    • Runs the Function Evaluation Analysis.
