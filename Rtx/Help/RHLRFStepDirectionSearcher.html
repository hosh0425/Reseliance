HLRF Step Direction Searcher

Class Name

  • RHLRFStepDirectionSearcher

Location in Objects Pane

  • Methods > Step Direction Searcher > iHLRF

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Output

  • This object guides the direction of steps in a sequence of iteration points to the design point, in a balanced manner.
  • For more information, see Der Kiureghian Zhang (1994).

Right-click Menu

Remove

  • Removes the object.
