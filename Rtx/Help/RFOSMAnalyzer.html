FOSM Analyzer

Class Name

  • RFOSMAnalyzer

Location in Objects Pane

  • Methods > Analyzer > FOSM

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Function

  • List of defined limit-state functions to analyze.

Print Correlation Matrix

  • Determines whether to print correlation matrix.

Print Covariance Matrix

  • Determines whether to print covariance matrix.

Output

  • Probability of failure

Right-click Menu

Remove

  • Removes the object.

Run Analysis

  • Runs the FOSM analyzer.
