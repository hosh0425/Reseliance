Regional Loss

Class Name

  • RRegionalLossModel

Location in Objects Pane

  • Models > Model > Consequence > Economic > Regional Loss

Model Description

Model Form

    $$l = \left( {\sum\limits_{k = 1}^K {{\eta _k} \cdot {A_k} \cdot {C_k}} } \right) \cdot \exp \left( { - r \cdot t} \right)$$ $$\ln \left( {{\eta _k}} \right) = \ln \left[ {{\vartheta _{1k}} \cdot \Phi \left( {{\vartheta _{2k}} \cdot Sa_2^{{\vartheta _{5k}}} + {\vartheta _{3k}} \cdot Sa_1^{{\vartheta _{5k}}} - {\vartheta _{4k}}} \right)} \right] + {\varepsilon _k}$$
  • For further information on this model, refer to Mahsuli, M., Haukaas, T. (2013) and Mahsuli, M., Haukaas, T. (2013).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Boundary Location List

  • List of location objects of the corner points of the region

Centroid Location

  • A location object that will take the region’s centroid coordinates.

Sa 1

  • \({S_{a_1}}\) = 5%-damped elastic spectral acceleration at the period of 1.0s at the region’s centroid.

Sa 2

  • \({S_{a_2}}\) = 5%-damped elastic spectral acceleration at the period of 0.3s at the region’s centroid.

Unit Area Cost List

  • \({C_k}\) = Unit area cost list for different zones within the region.

Area Ratio List

  • \({A_k}\) = Area ratio list, that is the ratio of different zone areas to the total area of the region.

Time

  • \({t}\) = Time object

Annual Interest Rate

  • \({r}\) = Annual Interest Rate

Theta 1 List

  • \({\vartheta_{1k}}\) = List of \({\vartheta_{1}}\) model parameters for different zones within the region.

Theta 2 List

  • \({\vartheta_{2k}}\) = List of \({\vartheta_{2}}\) model parameters for different zones within the region.

Theta 3 List

  • \({\vartheta_{3k}}\) = List of \({\vartheta_{3}}\) model parameters for different zones within the region.

Theta 4 List

  • \({\vartheta_{4k}}\) = List of \({\vartheta_{4}}\) model parameters for different zones within the region.

Theta 5 List

  • \({\vartheta_{5k}}\) = List of \({\vartheta_{5}}\) model parameters for different zones within the region.

Epsilon List

  • \({\varepsilon_{k}}\) = List of \({\varepsilon}\) model error for different zones within the region.

Color

  • Color of the region on Google Maps

Total Area

  • Total Area of the region

Output

  • \({I}\) = Total regional loss due to repair cost of buildings.
  • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.

  • Displays the object on map.

Remove From Map

  • Removes the object from map.
