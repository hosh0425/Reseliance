Time

Class Name

  • RTime

Location in Objects Pane

  • Models > Parameter > Time

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Current Value

  • The value of Current time in terms of days

Current Time

  • Current time in terms of Year-Month-Day Hour:Minute:Second

Output

  • Current time value

Right-click Menu

Remove

  • Removes the object.
