Water Station Serviceability

Class Name

  • RWaterStationServiceabilityModel

Location in Objects Pane

  • Models > Model > Infrastructure > Water > Water Station Serviceability

Model Description

Model Form

    $$S=\left\{ \begin{matrix} 1 & ; & \left( F =1 \right)\text{AND}\left( P=1 \right) \\ 0 & ; & \text{Otherwise} \\ \end{matrix} \right.$$

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Water Station Information Model

  • The associate Water Station information model

Functionality

  • \({F}\) = Status of functionality of the Water station (1 if functional, 0 otherwise)

Power Serviceability

  • \({P}\) = The serviceability status of the upstream electricity provider (1 if serviceable, 0 otherwise)

Output

  • \({S}\) = Serviceability status of the Water station (1 if serviceable, 0 otherwise).
    • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.
    Note: Serviceability features the capability of the water station to pump water.

Right-click Menu

Remove

  • Removes the object.
