Healthcare Demand Model

Class Name

  • RHealthcareDemandModel

Location in Objects Pane

  • Models > Model > Infrastructure > Healthcare > Healthcare Demand

Model Description

Model Form

    \[{{N}_{all}}=\sum{{{n}_{i}}}\] Note1: The "All" response, i.e. \({N_{all}}\), enumerates the total number of injuries who have been rescued since the start of the analysis (until the time the model is called).
    Note2: The "New" response, i.e. \({N_{new}}\), is the same as "All" response with the difference that it enumerates the number of injuries who have been just rescued and added to the healthcare demand (at the time the model is called).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Number Of Rescued Injuries List

  • {\({n_i}\)} = Number of rescued injuries for building \({i}\) (or any other facility which produces casualties due to the hazard)

Output

  • \({N_{all}}\) = Total number of injuries who need to be hospitalized
    • The output is an automatically generated generic response object, which takes the object name of the model plus “AllResponse”.
  • \({N_{new}}\) = The number of injuries who have been just rescued and need to be hospitalized (at the time the model is called).
    • The output is an automatically generated generic response object, which takes the object name of the model plus “NewResponse”.

Right-click Menu

Remove

  • Removes the object.
