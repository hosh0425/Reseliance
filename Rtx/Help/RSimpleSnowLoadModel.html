Simple Snow Load Model

Class Name

  • RSimpleSnowLoadModel

Location in Objects Pane

  • Models > Model > Hazard > Snow > Simple Snow Load

Model Description

Model Form

    $$\eqalign{ & d = {{{d_p}} \over 2} \cdot \left( {1 + \sin \left( {2\pi \cdot {{t - {m_s}} \over {{m_e} - {m_s}}} - {\pi \over 2}} \right)} \right) \cdot \varepsilon \cr & q = \rho \cdot d \cr} $$
  • For further information on this model, refer to Mahsuli, M., Haukaas, T. (2013).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Maximum Depth

  • \({d_p}\) = Peak seasonal snow depth

Start Month

  • \({m_s}\) = End month of the snow season (given as a real number)

End Month

  • \({m_e}\) = End month of the snow season (given as a real number)

Density

  • \({\rho}\) = Snow density

Time

  • \({t}\) = Current time, typically a Time parameter

Epsilon

  • \({\varepsilon}\) = Model error

Output

  • \({d}\) = Snow depth
  • \({q}\) = Snow load per unit area
  • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.

References

  • Mahsuli, M., Haukaas, T. (2013) “Computer program for multimodel reliability and optimization analysis,” ASCE Journal of Computing in Civil Engineering, 27(1), pp. 87-98 (DOI)
