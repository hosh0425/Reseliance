Commercial Information Model

Class Name

  • RCommercialInformationModel

Location in Objects Pane

  • Models > Model > Infrastructure > Commercial > Commercial Information

Model Description

Model Form

  • This model is a database of Commercial information.
  • The model is passed to any other model that needs the Commercial information.

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Building Information Model

  • The associate building information model

Time of Establishment

  • A time object signifying the establishment time

Number Of Employees

  • The number of employees who work at this commercial unit

Sector

  • Sector of the Commercial unit (Locally Oriented or Export Oriented.)

Size

  • Size of the Commercial unit (Small or Large.)

Output

  • This model does not generate any responses.

Right-click Menu

Remove

  • Removes the object.
