Building Utility Model

Class Name

  • RBuildingUtilityModel

Location in Objects Pane

  • Models > Model > Infrastructure > Building > Building Utility Model

Model Description

Model Form

    $$ \varsigma_p = p$$ $$ \varsigma_w = w$$

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Building Information Model

  • The associate building information model

Power Serviceability

  • \({p}\) = The serviceability status of the upstream electricity provider

Water Serviceability

  • \({w}\) = The serviceability status of the upstream water provider

Output

  • \({\varsigma_p}\) = Determines the status of electricity of the building (1 if yes, 0 otherwise)
    • The output is an automatically generated generic response object, which takes the object name of the model plus “PowerResponse”.
  • \({\varsigma_w}\) = Determines the status of water of the building (1 if yes, 0 otherwise)
    • The output is an automatically generated generic response object, which takes the object name of the model plus “WaterResponse”.

Right-click Menu

Remove

  • Removes the object.
