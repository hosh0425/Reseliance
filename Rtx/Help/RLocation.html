Location

Class Name

  • RLocation

Location in Objects Pane

  • Models > Parameter > Location

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Latitude

  • Latitude of the location

Longitude

  • Longitude of the location

Altitude

  • Altitude of the location

Address

  • Address of the location
  • This option is only applicable in Canada.

Output

  • Geographical coordinates of a location

Right-click Menu

Remove

  • Removes the object.

Display On Map

  • Displays the locations on map.

Remove From Map

  • Removes the locations on map.

Get Coordinates From Map

  • Gets the coordinates of a location by clicking on map.

Get Coordinates From Address

  • Gets the coordinates of a location using address.
  • This option is only applicable in Canada.
