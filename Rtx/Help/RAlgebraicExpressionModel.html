Algebraic Expression Model

Class Name

  • RAlgebraicExpressionModel

Location in Objects Pane

  • Models > Model > Generic > Algebraic Expression

Model Description

Model Form

  • This model evaluates a mathematical expression.

DDM Sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages in the Output Pane.

Expression

  • The mathematical expression to be evaluated

Output

  • The calculated value of mathematical expression
  • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.
