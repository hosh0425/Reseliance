Discrete Event Sampling Analyzer

Class Name

  • RDiscreteEventSamplingAnalyzer

Location in Objects Pane

  • Methods > Analyzer > Discrete Event Sampling

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Output Display Level

  • Determines the level of messages and outputs of this object to print in the Output Pane.
  • The higher the level of outputs, the more time it takes to run the analysis.
  • The “None” option, prints nothing.
  • The “Minimum” option, prints the finish time of the current event and the function values.
  • The “Moderate” option, prints the finish time of the current event, the updated parameters, and the function values.
  • The “Maximum” option, prints the finish and start time of the current event, the updated parameters, and the function values

Display Diagrams

  • Determines whether the output is allowed to plot diagrams. If "True", four different types of diagrams are generated.
  • Note: This property is automatically turned off and set to false for multi-threaded analysis, i.e. no diagram is plotted.

Function Estimation Interval

  • The interval (in days) which the function values are reported on. For example, if the interval is 7 days, the function values are reported on days 7, 14, 21, and so on.

Random Number Generator

  • The defined random number generator

Transformer

  • Indicates the transformer object which transforms random variables in original space to the standard normal space or vice versa

Accumulator

  • The defined accumulator

Function List

  • List of defined functions to analyze

Function Accumulation Type List

  • Determines the generated output files, containing the time history information of the PDF and CDF of the Incremental and Cumulative values of functions.
  • The “0” option, no output file is generated.
  • The “1” option, two output files are generated for each function, containing the time history information of the PDF and CDF of the incremental values of functions.
  • The “2” option, two output files are generated for each function, containing the time history information of the PDF and CDF of the cumulative values of functions.
  • The “3” option, four output files are generated for each function, containing the time history information of the PDF and CDF of the incremental and cumulative values of functions.

Occurrence Model List

  • List of defined occurrence models
  • For each sample, one hazard is selected and others are turned off. This selection is based of the occurrence rate of the hazards.

Event Model List

  • List of defined event models

Time Parameter

  • Time object.

Output

  • The cumulative values of functions at the end of the simulation.

Right-click Menu

Remove

  • Removes the object.

Run Analysis

  • Runs the scenario sampling analyzer.
