Power Station Damage Model

Class Name

  • RPowerStationDamageModel

Location in Objects Pane

  • Models > Model > Infrastructure > Power > Power Station Damage

Model Description

Model Form

  • This model produces the damage ratio and damage state for a power station, subjected to the ground shaking based on the fragility curves provided by HAZUS (FEMA-NIBS 2012) approach.

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Power Station Information Model

  • The associate Power Station information model

Intensity

  • Peak Ground Acceleration (PGA) at the power station site

Damage State Uncertainty

  • A standard normal random variable, i.e. N(0,1), which captures the uncertainty in damage state for a PGA.

Damage Ratio Uncertainty

  • A standard normal random variable, i.e. N(0,1), which captures the uncertainty in damage ratio for a damage state.

Output

  • \({d}\) = Damage state of the power station
    • The output is an automatically generated generic response object, which takes the object name of the model plus “StateResponse”.
  • \({\eta}\) = Damage ratio of the power station
    • The output is an automatically generated generic response object, which takes the object name of the model plus “RatioResponse”.

Right-click Menu

Remove

  • Removes the object.
