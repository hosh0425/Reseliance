Load Coincidence Analyzer

Class Name

  • RLoadCoincidenceAnalyzer

Location in Objects Pane

  • Methods > Analyzer > Load Coincidence

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Limit State Function

  • List of defined limit-state functions to analyze

Analysis Type

  • List of the types of analysis including FORM, sampling and importance sampling

FORM Analyzer

  • List of defined FORM analyzers

Sampling Analyzer

  • List of defined Sampling analyzers

Occurrence Model List

  • List of defined occurrence models

Max Number Of Coincident Hazards

  • Determines maximum number of individual hazards to take into account for the occurrence and coincidence of multiple hazards.

Timespan

  • Life time
  • The standard unit of lifetime is defined in terms of days.

Compute Random Variable Sensitivities

  • Determines whether to compute random variable sesitivities.

Compute Decision Variable Sensitivities

  • Determines whether to compute decision variable sesitivities.

Compute Model Response Standard Deviation Sensitivities

  • Determines whether to compute model response standard deviation sensitivities.

Print Sensitivities

  • Determines whether to print sensitivities.

Correct Probability With First Principal Curvature

  • Determines whether to correct probability with first principal curvature.

Start Point File

  • Absolute or relative path to the location of the start point file

Design Point File

  • Absolute or relative path to the location of the design point file

Output

  • The lifetime probability of failure

Right-click Menu

Remove

  • Removes the object.

Run Analysis

  • Runs the load coincidence analyzer.
