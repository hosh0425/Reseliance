Continous Decision Variable

Class Name

  • RContinuousDecisionVariable

Location in Objects Pane

  • Models > Parameter > Decision Variable > Continuous

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Current Value

  • Current realization of the decision variable

Initial Value

  • Initial value of the decision variable

Upper Bound

  • Upper bound of the decision variable

Lower Bound

  • Lower bound of the decision variable

Incremental Cost

  • Incremental cost of the decision variable

Read-only Properties

Beta Sensitivity Wrt Cost

  • Derivative of the reliability index, \({\beta}\), with respect to the cost of the decision variable

Pf Sensitivity Wrt Cost

  • Derivative of the probability of the failure, \({P_f}\), with respect to the cost of the decision variable

Output

  • Current value

Right-click Menu

Remove

  • Removes the object.
