Healthcare Information Model

Class Name

  • RHealthcareInformationModel

Location in Objects Pane

  • Models > Model > Infrastructure > Healthcare > Healthcare Information

Model Description

Model Form

  • This model is a database of Healthcare information.
  • The model is passed to any other model that needs the Healthcare information.

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Building Information Model

  • The associate building information model

Number Of Beds

  • The hospitalization capacity of the helthcare unit in terms of its number of beds.

Sector

  • Sector of the Healthcare unit (Public or Private.)

Size

  • Size of the Healthcare unit (Small, Medium, or Large.)

Type

  • Type of the Healthcare unit (Hospital or Medical Office/Clinic.)

Output

  • This model does not generate any responses.

Right-click Menu

Remove

  • Removes the object.
