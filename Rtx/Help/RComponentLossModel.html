Component Loss Model

Class Name

  • RComponentLossModel

Location in Objects Pane

  • Models > Model > Consequence > Economic > Component Loss Model

Model Description

Model Form

    This model computes the total loss of components of the same type by considering bulk discount as well. The following model computes the total loss of the components of the same type: $$ \eqalign{ &{C_{comp}}(\eta) = \sum ^{N_C}_{i=1} \sum ^{N_M}_{j=1} ((R_j(\eta_i)-R_{j+1}(\eta_i))\cdot(1-D_j(n_j))\cdot\eta_i \cdot C_{max,comp} \cdot Q_i) }$$ where, \(N_C\) is total number of components of the desired type, \(N_M\) is number of methods of repair and \(n_j\) is number of the components of the same type require the \(j^{th}\) method of repair.
  • For further information on this model, refer to Aghababaei, M., Mahsuli, M. (2019) and Aghababaei, M., Mahsuli, M. (2018).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rtx
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”)
  • The name is unique and case-sensitive

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane

Damage Ratio List

  • List of the damage ratios for all components for which this model computes the loss
  • \(\eta_i\) in the formula above is the damage ratio in component i

\(R_{j}\)

    \[{{R}_{j}}({{\eta }_{i}})=\left\{ \begin{array}{*{35}{l}} 0 & {{\eta }_{i}}\le {{\eta }_{Tj}}-\Delta {{\eta }_{Tj}} \\ \frac{1}{2}\sin \left( \frac{\pi }{2}\frac{{{\eta }_{i}}-{{\eta }_{Tj}}}{\Delta {{\eta }_{Tj}}} \right)+\frac{1}{2} & {{\eta }_{Tj}}-\Delta {{\eta }_{Tj}} <{{\eta }_{i}}<{{\eta }_{Tj}}+\Delta {{\eta }_{Tj}} \\ 1 & {{\eta }_{i}}\ge {{\eta }_{Tj}}+\Delta {{\eta }_{Tj}}\\ \end{array} \right.\]

Threshold Damage Ratio

  • \(\eta_{Tj}\) = damage ratio threshold after which the \(j^{th}\) or higher methods of repair are required; \(η_{Tj} − ∆ η_{Tj}\) and \(η_{Tj} + ∆ η_{Tj}\) for all methods of repair are entered as a list in Rtx in “Threshold Damage Ratios.” Hence, the length of the list is twice the number of methods of repair.

Discount Rates

  • \(D_j\) = discount ratio for method of repair j
    • \[{{D}_{j}}({{n}_{j}})=\left\{ \begin{array}{*{35}{l}} 0 & {{n}_{j}}\le {{n}_{lj}} \\ 0.5D_{j}^{\max }\cdot \left( 1-\cos \left( \pi \frac{{{n}_{j}}-{{n}_{lj}}}{{{n}_{uj}}-{{n}_{lj}}} \right) \right) & {{n}_{lj}} <{{n}_{j}}<{{n}_{uj}} \\ D_{j}^{\max } & {{n}_{j}}\ge {{n}_{uj}} \\ \end{array} \right.\]

Lower Quantity

  • \(n_{lj}\) = the quantity lower than which there is zero discount. Lower quantities for all methods of repair are entered in Rtx in “Lower Quantity” as a list

Upper Quantity

  • \(n_{uj}\) = the quantity more than which there is Djmax discount. Upper quantities for all methods of repair are entered in Rtx in “Upper Quantity” as a list

Discount Rates

  • \(D_{j^{max}}\) = the maximum discount rate \(D_{j^{max}}\) when \(n_{j}\) is larger than the threshold \(n_{j}\)

Replacement Cost Per Unit

  • \(C_{max,comp}\) = Maximum unit cost of repair for the desired component type, usually, the unit cost of replacement

Areas

  • \(Q_i\)=quantity of the \(i^{th}\) component, e.g., the area of a partition wall

Area Dependent

  • in Rtx using the property “Area Dependent” one can determine whether or not the component is dependent to area. If yes, the area which defines area of one quantity in computing discount should be entered.

Output (Automatically generated generic response object)

  • \(C_{comp}\) = total repair cost of the components of the same type
