Rtx Overview

Introduction

  • Rtx (Reliability Tool) is a state-of-the-art reliability software application developed at the University of British Columbia as part of the Infrastructure Risk Project (www.inrisk.ubc.ca).
  • Rtx is freely available for download for Microsoft Windows, Mac OS X and Linux operating systems.
  • The software provides a flexible, convenient and powerful interface to implement probabilistic models, orchestrate analyses that utilize several models, evaluate limit-state functions to calculate their probabilities of exceedance using reliability analysis algorithms, communicate with external software, handle and visually display spatial coordinates on a map, plot histograms through sampling algorithms and find importance vectors of modeled parameters.
