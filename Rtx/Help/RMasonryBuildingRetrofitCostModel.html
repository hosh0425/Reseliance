Model Description

Model Form

    ////
    Important Notes:
  • The cost of retrofit operation is computed once it is completed (i.e. the retrofit status changes to 1), otherwise it is 0.
  • To use this model in analyzers other than Discrete Event Sampling Anlyzer, the "RetrofitStatus" property must be a constant parameter with current value of 1
  • For further information on this model, refer to Nasrazadani, H., Mahsuli, M., Talebiyan, H., Kashani, H. (2017).
