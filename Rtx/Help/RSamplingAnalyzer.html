Sampling Analyzer

Class Name

  • RSamplingAnalyzer

Location in Objects Pane

  • Methods > Analyzer > Sampling

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Random Number Generator

  • List of defined random number generators

Transformer

  • Indicates the transformer object which transforms random variables from original space to the standard normal space or vice versa.

Accumulator

  • List of defined accumulators

Function List

  • List of defined functions to analyze

Output

  • Probability of failure

Right-click Menu

Remove

  • Removes the object.

Run Analysis

  • Runs the Sampling analyzer.
