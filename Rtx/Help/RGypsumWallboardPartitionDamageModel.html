Gypsum Wallboard Partition Damage Model

Class Name

  • RGypsumWallboardPartitionDamageModel

Location in Objects Pane

  • Models > Model > Consequence > Economic > Gypsum Wallboard Partition Damage

Model Description

Model Form

    This model predicts damage ratio in gypsum wallboard partitions: $$ \ln (-\ln (1-{\eta})) = -1.47 + {\theta_1} \cdot {I_{RW}}+{\theta_2} \cdot (\log(\delta)+2.26)+\epsilon \cdot \sigma$$
  • For further information on this model, refer to Aghababaei, M., Mahsuli, M. (2019) and Aghababaei, M., Mahsuli, M. (2018).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Has Return Wall

  • \(I_{RW}\) = equals unity if the partitions has return walls at the ends and zero otherwise

Inter-story Drift Ratio

  • \({\delta}\) = inter-story drift ratio

Theta1

  • \({\theta_1}\) = Model parameter

Theta2

  • \({\theta_2}\) = Model parameter

Sigma

  • \({\sigma}\) = Standard deviation of the model error

Epsilon

  • \({\varepsilon}\) = model error; a standard normal random variable

Output (Automatically generated generic response object)

  • \({\eta}\) = Damage ratio

Right-click Menu

Remove

  • Removes the object.
