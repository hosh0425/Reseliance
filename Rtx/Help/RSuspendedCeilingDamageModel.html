Suspended Ceiling Damage Model

Class Name

  • RSuspendedCeilingDamageModel

Location in Objects Pane

  • Models > Model > Consequence > Economic > Suspended Ceiling Damage

Model Description

Model Form

    This model predicts damage ratio in suspended ceiling: $$ {\Phi^{-1}}(\eta) = 0.5655 - 0.0207(\ln(A)-6.544) +{\theta_1}.(\ln({a_f}) + 0.50714(\ln(A) - 6.544)) + {\theta_2}.{I_{LS}} + \epsilon\cdot \sigma$$ where, \({\Phi^{-1}}\) is the probit function.
  • For further information on this model, refer to Aghababaei, M., Mahsuli, M. (2019) and Aghababaei, M., Mahsuli, M. (2018).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Ceiling Area

  • \(A\) = Total area of ceiling

Peak Floor Acc

  • \({a_f}\) = Floor acceleration demand

Hast Lateral Support

  • \({I_{LS}}\) = binary variable that equals unity if the ceiling has lateral support and zero otherwise

Theta1

  • \({\theta_1}\) = Model parameter

Theta2

  • \({\theta_2}\) = Model parameter

Sigma

  • \({\sigma}\) = Standard deviation of the model error

Epsilon

  • \({\varepsilon}\) = model error; a standard normal random variable

Output (Automatically generated generic response object)

  • \({\eta}\) = Damage ratio

Right-click Menu

Remove

  • Removes the object.
