Simple Wind Pressure Model

Class Name

  • RSimpleWindPressureModel

Location in Objects Pane

  • Models > Model > Hazard > Wind > Simple Wind Pressure

Model Description

Model Form

    $$q = {1 \over 2}\rho \cdot {V^2} \cdot {C_e} \cdot {C_g} \cdot {C_p} \cdot \varepsilon $$
  • For further information on this model, refer to Mahsuli, M., Haukaas, T. (2013).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Velocity

  • \({V}\) = Wind velocity

Air Density

  • \({\rho}\) = Air density

Exposure Factor

  • \({C_e}\) = Exposure factor

External Pressure Factor

  • \({C_p}\) = External pressure factor

Gust Factor

  • \({C_g}\) = Gust factor

Epsilon

  • \({\varepsilon}\) = Model error

Output

  • \({q}\) = Wind pressure
  • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.

References

  • Mahsuli, M., Haukaas, T. (2013) “Computer program for multimodel reliability and optimization analysis,” ASCE Journal of Computing in Civil Engineering, 27(1), pp. 87-98 (DOI)
