Building Search And Rescue Time Model

Class Name

  • RBuildingSearchAndRescueTimeModel

Location in Objects Pane

  • Models > Model > Infrastructure > Building > Building Search And Rescue Time

Model Description

Model Form

  • For a collapsed building, the duration of search and rescue operation is computed as below
    • $$\tau ={{t}_{r}}\cdot A\cdot {{\eta }_{S}}$$ where the total floor area, “\({A}\)”, is obtained from the Building information model.
    Important Notes:
  • The building is collapsed if the structural damage ratio exceeds 0.8, i.e. \({\eta _S}\)>0.8, and the value of the "Collapse Uncertainty", \({\beta_{col}}\), is less than the collapse probability.
  • The collapse probability of the building is obtained from HAZUS (FEMA-NIBS 2012), which depends on the building's "Building Type", i.e. \({B}\).
  • If the building is not collapsed it is assumed that no search and rescue operation is required, i.e. \({\tau}\) = 0.
  • The status response is computed as below.
    • \[\varsigma =\left\{ \begin{matrix} 0 & ; & \tau >0 \\ 1 & ; & \tau =0 \\ \end{matrix} \right.\]

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Building Information Model

  • The associate building information model

Building Type

  • \({B}\) = Building type per HAZUS (FEMA-NIBS 2012)

Structural Damage Ratio

  • \({\eta _S}\) = Structural Damage Ratio

Search And Rescue Rate

  • \(t_r\) = Duration of search and rescue operation for one squared meter of the building's total floor area

Collapse Uncertainty

  • \({\beta_{col}}\) = a standard normal random variable, i.e. N(0,1)

Output

  • \({\tau}\) = Duration of search and rescue operation of the building
    • The output is an automatically generated generic response object, which takes the object name of the model plus “TimeResponse”.
  • \({\varsigma}\) = Determines the status of search and rescue operation (1 if completed, 0 otherwise). Once the scheduled operation is completed, the status changes to 1
    • The output is an automatically generated generic response object, which takes the object name of the model plus “StatusResponse”.
