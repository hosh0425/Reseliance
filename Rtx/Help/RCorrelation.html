Correlation

Class Name

  • RCorrelation

Location in Objects Pane

  • Models > Parameter > Correlation

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Random Variable 1

  • First random variable from all random variables, whose correlation with another random variable is about to be set

Random Variable 2

  • Second random variable from all random variables, whose correlation with another random variable is about to be set

Coefficient

  • Correlation Coefficient (\(\boldsymbol{\rho}\)) between Random Variable 1 and Random Variable 2

Output

  • Correlation Coefficient

Right-click Menu

Remove

  • Removes the object.
