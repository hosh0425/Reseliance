Building Search And Rescue Cost Model

Class Name

  • RBuildingSearchAndRescueCostModel

Location in Objects Pane

  • Models > Model > Infrastructure > Building > Building Search And Rescue Cost

Model Description

Model Form

    $$C = c_r \cdot t_{sar} $$ Important Notes:
  • The cost of search and rescue operation is computed once it is completed (i.e. the search and recue status changes to 1), otherwise it is 0.
  • To use this model in analyzers other than Discrete Event Sampling Anlyzer, the "SearchAndRescueStatus" property must be a constant parameter with current value of 1

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Building Information Model

  • The associate building information model

Search And Rescue Cost Rate

  • \({c_r}\) = Cost of search and rescue operation for one day

Structural Damage Ratio

  • \({\eta _S}\) = Structural Damage Ratio

Search And Rescue Status

  • \({\varsigma}\) = Status of Search And Rescue operation (1 if completed, 0 otherwise)

Search And Rescue Time

  • \({t_{sar}}\) = duration of search and rescue operation of the building

Output

  • \({C}\) = Cost of search and rescue operation of the building
    • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.
