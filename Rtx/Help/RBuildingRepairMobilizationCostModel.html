Building Repair Mobilization Cost Model

Class Name

  • RBuildingRepairMobilizationCostModel

Location in Objects Pane

  • Models > Model > Infrastructure > Building > Building Repair Mobilization Cost

Model Description

Model Form

    $$C = c \cdot \eta _S $$ Important Notes:
  • The cost of mobilization operation is computed once it is completed (i.e. the mobilization status changes to 1), otherwise it is 0.
  • To use this model in analyzers other than Discrete Event Sampling Anlyzer, the "MobilizationStatus" property must be a constant parameter with current value of 1

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Building Information Model

  • The associate building information model

Structural Damage Ratio

  • \({\eta _S}\) = Structural Damage Ratio

Mobilization Cost Parameter

  • \({c}\) = Cost of mobilizing the repair of the building in complete damage state (input by user)

Mobilization Status

  • \({\varsigma}\) = Status of Mobilization operation (1 if completed, 0 otherwise)

Output

  • \({C}\) = Cost of mobilizng the repair of the building
    • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.
