Water Station Repair Time Model

Class Name

  • RWaterStationRepairTimeModel

Location in Objects Pane

  • Models > Model > Infrastructure > Water > Water Station Repair Time

Model Description

Model Form

  • This model produces the repair time of a Water Station, \({\tau}\), subjected to ground shaking based on the HAZUS (FEMA-NIBS 2012) approach.
    • Note: If the repair time of a Water Station is less than one day, i.e. \({\tau}\) < 1, it is assumed to be 0, i.e. \({\tau}\) = 0.

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Water Station Information Model

  • The associate Water Station information model

Damage Ratio

  • \({\eta}\) = Damage Ratio

Repair Time Uncertainty

  • A standard normal random variable, i.e. N(0,1), which captures the uncertainty in repair time for a damage ratio.

Output

  • \({\tau}\) = Duration of the repair of the Water Station
    • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.
