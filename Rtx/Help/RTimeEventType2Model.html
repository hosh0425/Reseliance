Time Event Type 2

Class Name

  • RTimeEventType2Model

Location in Objects Pane

  • Models > Model > Event > Time Type 2

Model Description

Model Form

  • This model schedules the events in some specified points in time, input by the user.
    • Note: At those specified points in time, based on the percentage values input by the user, a fraction of current value of parameter 1 and parameter 2 is substracted and added to the corresponding current value of parameter 3.

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Time Parameter

  • Time object

Event Time List

  • List of the time of the events which are supposed to be scheduled.

Parameter List 1

  • 1st List of parameters which are supposed to be modifed at the time of the scheduled events.

Parameter List 2

  • 2nd List of parameters which are supposed to be modifed at the time of the scheduled events.

Parameter List 3

  • 3rd List of parameters which are supposed to be modifed at the time of the scheduled events.

Percentages Of Parameter 1 Added To Parameter 3 List

  • List of the percentage values which, upon the occurence of the event, are supposed to be substracted from the current value of parameter 1 and added to the current value of corresponding parameter 3. This size of this list must be either one or equal to the size of the list of parameters.

Percentages Of Parameter 2 Added To Parameter 3 List

  • List of the percentage values which, upon the occurence of the event, are supposed to be substracted from the current value of parameter 2 and added to the current value of corresponding parameter 3. This size of this list must be either one or equal to the size of the list of parameters.

Output

  • This model does not generate response objects.
  • It provides the scheduled events to the orchestrating algorithm.

Right-click Menu

Remove

  • Removes the object.
