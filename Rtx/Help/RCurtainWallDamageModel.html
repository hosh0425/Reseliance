Curtain Wall Damage Model

Class Name

  • RCurtainWallDamageModel

Location in Objects Pane

  • Models > Model > Consequence > Economic > Curtain Wall Damage

Model Description

Model Form

    This model predicts damage ratio in Curtain Wall: $$ \eqalign{ &{\Phi^{-1}}(\eta) = \cr & - 0.007d + 4.117(ln(1+d)-0.17276d) \cr & + 0.12({I_{AIGU}}+0.812({I_{IGU}}-0.62(\ln(1+d)-0.17276d))) \cr & - 0.825({I_{mon}}+1.0214({I_{AIGU}}+0.812({I_{IGU}}-0.62(\ln(1+d)-0.17276d)))) \cr & - 0.47(\ln(\delta)+6.393({I_{mon}}+1.0214({I_{AIGU}}+0.812({I_{IGU}}-0.62(\ln(1+d)-0.17276d))))) \cr & + {\theta_1}.({I_{lam}}-0.688(\ln(\delta)+6.393({I_{mon}}+1.0214({I_{AIGU}}+0.812({I_{IGU}}-0.62(\ln(1+d)-0.17276d)))))) \cr & + {\epsilon \cdot \sigma} }$$
  • For further information on this model, refer to Aghababaei, M., Mahsuli, M. (2019) and Aghababaei, M., Mahsuli, M. (2018).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Glass to Frame Clearance

  • \(d\) = Glass to frame clearance

Laminated Glass

  • \({I_{lam}}\) = Unity if the glass is laminated and zero otherwise

Glass Type

  • \({I_{IGU}}\) = Unity if the glass is IGU and zero otherwise
  • \({I_{AIGU}}\) = Unity if the glass is AIGU and zero otherwise
  • \({I_{mon}}\) = Unity if the glass is monolithic and zero otherwise

Inter-story Drift Ratio

  • \({\delta}\) = Inter-story drift ratio

Theta1

  • \({\theta_1}\) = Model parameter

Sigma

  • \({\sigma}\) = Standard deviation of the model error

Epsilon

  • \({\varepsilon}\) = model error; a standard normal random variable

Output (Automatically generated generic response object)

  • \({\eta}\) = Damage ratio

Remove

  • Removes the object.
