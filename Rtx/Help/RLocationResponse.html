Location Response

Class Name

  • RLocationResponse

Location in Objects Pane

  • Models > Parameter > Response > Location

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Current Value

  • Current realization of the model response

Model

  • List of the models

Location

  • List of the locations

Read-only Properties

Total Standard Deviation

  • Total standard deviation of location response object

Beta Sensitivity Wrt Total Standard Deviation

  • Derivative of the reliability index, \({\beta}\), with respect to the total standard deviation of this location response object

Total Epistemic Standard Deviation

  • Total epistemic standard deviation of location response object

Beta Sensitivity Wrt Total Epistemic Standard Deviation

  • Derivative of the reliability index, \({\beta}\), with respect to the total epistemic standard deviation of this location response object

Model Standard Deviation

  • Standard deviation of the model used to extract location response object

Beta Sensitivity Wrt Model Standard Deviation

  • Derivative of the reliability index, \({\beta}\), with respect to the model standard deviation of this location response object

Model Epistemic Standard Deviation

  • Epistemic standard deviation of the model used to extract location response object

Beta Sensitivity Wrt Model Epistemic Standard Deviation

  • Derivative of the reliability index, \({\beta}\), with respect to the model epistemic standard deviation of this location response object

Output

  • Current value
  • The output is automatically generated as location object, which takes the object name of the object plus “Location”.

Right-click Menu

Remove

  • Removes the object.
