Rectangular Area Source Model

Class Name

  • RRectangularAreaSourceModel

Location in Objects Pane

  • Models > Model > Hazard > Earthquake > Location > Recatangular Area Source

Model Description

Model Form

  • This model produces a location within a rectangular area that is defined by two diagonal corner points.
  • The sides of the area align with Earth’s latitudes and longitudes.
  • For further information on this model, refer to Rahimi, H., Mahsuli, M. (2019) and Mahsuli, M., Rahimi, H., Bakhshi, A. (2019).

DDM Sensitivities

  • Yes

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Point 1 Location

  • Location object for one corner of the area

Point 2 Location

  • Location object for the diagonally opposite corner

Theta 1

  • \({\vartheta_1}\) = Value between 0 and 1 that defines the output longitude, typically a random variable uniformly distributed between 0 and 1.

Theta 2

  • \({\vartheta_2}\) = Value between 0 and 1 that defines the output latitude, typically a random variable uniformly distributed between 0 and 1.

Depth

  • Depth of the output location, in km

Output

  • Location object carried by a location response object.
  • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.

Display On Map

  • Displays the object on map.

Remove From Map

  • Removes the object from map.

References
