Power Station Information Model

Class Name

  • RPowerStationInformationModel

Location in Objects Pane

  • Models > Model > Infrastructure > Power > Power Station Information

Model Description

Model Form

  • This model is a database of power station information.
  • The model is passed to any other model that needs the power station information.

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Location

  • Location of the power station

Time of Construction

  • A time object signifying the construction time

Replacement Cost

  • Cost of replacing the power station

Type

  • Type of the power station (High Voltage, Medium Voltage, or Low Voltage.)

Components

  • Determines whether the components are anchored to the ground or not

Output

  • This model does not generate any responses.

Right-click Menu

Remove

  • Removes the object.
