Population Density Model

Class Name

  • RPopulationDensityModel

Location in Objects Pane

  • Model > Consequence > Social > Population Density Model

Model Description

Model Form

    The population density is computed as the maximum yearly population density times three coefficients that respectively account for the reduction in the population based on the hour of the day, the day of the week, and the month of the year in which the earthquake occurs.
  • For further information on this model, refer to Aghababaei, M., Mahsuli, M. (2019) and Aghababaei, M., Mahsuli, M. (2018).

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rtx
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”)
  • The name is unique and case-sensitive

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane

Time

  • Time of occurrence of the event

Maximum Population Density

  • Maximum yearly population density

Population Ratios Per Month

  • A vector of 12 elements, each of which is a number between 0 and 1, showing the monthly change in the population

Population Ratios Per Weekday

  • A vector of 7 elements, each of which is a number between 0 and 1, showing the daily change in the population during a week

Population Ratios Per Hour

  • A vector of 24 elements, each of which is a number between 0 and 1, showing the hourly change in the population during a day.

Output (Automatically generated generic response object)

  • The population density at the time of earthquake occurrence

Right-click Menu

Remove

  • Removes the object.
