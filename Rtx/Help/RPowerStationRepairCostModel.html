Power Station Repair Cost Model

Class Name

  • RPowerStationRepairCostModel

Location in Objects Pane

  • Models > Model > Infrastructure > Power > Power Station Repair Cost

Model Description

Model Form

    $$C = c \cdot \eta $$ Important Notes:
  • The cost of repair operation is computed once it is completed (i.e. the serviceability changes to 1), otherwise it is 0.
  • To use this model in analyzers other than Discrete Event Sampling Anlyzer, the "PowerStationServiceability" property must be a constant parameter with current value of 1

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Power Station Information Model

  • The associate Power Station information model

Damage Ratio

  • \({\eta}\) = Damage Ratio

Power Station Serviceability

  • \({S}\) = Status of serviceability of the power station (1 if serviceable, 0 otherwise)

Replacement Value

  • \({c}\) = Cost of replacing the power station

Output

  • \({C}\) = Cost of repair of the Power Station
    • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.
