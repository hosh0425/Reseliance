Healthcare Treatment Cost Model

Class Name

  • RHealthcareTreatmentCostModel

Location in Objects Pane

  • Models > Model > Infrastructure > Healthcare > Healthcare Treatment Cost

Model Description

Model Form

    \[C=\sum\nolimits_{i}^{N}{{{c}_{i}}\cdot {{\rho }_{i}}}\]

DDM sensitivities

  • No

Properties

Object Name

  • Name of the object in Rt
  • Allowable characters are upper-case and lower-case letters, numbers, and underscore (“_”).
  • The name is unique and case-sensitive.

Display Output

  • Determines whether the model is allowed to print messages to the Output Pane.

Healthcare Information Model

  • The associate Healthcare information model

Number Of Injury Levels

  • \({N}\) = Number of injury severity levels, excluding uninjured and dead people.

Number Of Treated Injuries List

  • {\({\rho_i}\)} = Number of treated injuries in severity level \({i}\)

Unit Cost Of Treatment List

  • {\({c_i}\)} = treatment cost of an injured with injury severity level of \({i}\).

Output

  • \({C}\) = Cost of treatment of injured
    • The output is an automatically generated generic response object, which takes the object name of the model plus “Response”.

Right-click Menu

Remove

  • Removes the object.
